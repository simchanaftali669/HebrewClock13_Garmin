# JClock - Jerusalem Clock for Garmin

JClock is a Garmin watch application that presents time through a
Jerusalem-based conceptual clock.

Unlike a regular clock, which is mainly used for coordinating practical time,
JClock is built around the idea that time can also carry meaning. The app is
anchored to a fixed reference point in Jerusalem and translates each moment
into a unique time expression according to a defined calculation method.

## Overview

The purpose of JClock is to offer a different way of looking at time: not only
as hours, minutes, and seconds, but as a meaningful structure connected to
Jerusalem.

The Garmin version is designed to run alongside the official JavaScript-based
JClock implementation and to bring the same concept into a wearable daily-use
experience.

## What the App Does

JClock displays a time system based on a Jerusalem-centered calculation.

The app may also show the regular current time for convenience, but the main
focus is the JClock time expression.

The app is intended for users interested in:

- A symbolic and meaningful view of time
- Jerusalem-centered time concepts
- Hebrew calendar and spiritual time ideas
- Experimental and non-standard clock systems
- Personal reflection through wearable technology

## The Idea Behind JClock

Most clocks answer a practical question:

> What time is it?

JClock asks a deeper question:

> What does this moment mean?

The app is based on the concept that there can be one central time reference
connected to Jerusalem. From that point, every moment can be represented
through a calculated expression.

In this sense, the app is not meant to replace the ordinary clock, but to stand
beside it and give the user another layer of awareness.

## Jerusalem as the Reference Point

JClock is not a location-changing clock. Its core reference is Jerusalem.

Even if the user is physically located elsewhere, the clock is conceptually
anchored to Jerusalem. This makes JClock a universal clock with one fixed point
of meaning rather than a local clock that changes according to the user's
current position.

## Main Features

- Jerusalem-based time concept
- Garmin watch display
- Optional display of regular time for convenience
- Custom calculated time expression
- Parallel concept to the official JavaScript clock at [JClock.net](https://JClock.net/)
- Lightweight interface designed for daily use
- Reflective and meaningful wearable experience

## How to Use

After installing the app on a supported Garmin device:

1. Open JClock from your Garmin apps list.
2. View the regular time if displayed.
3. View the JClock time expression.
4. Use the display as a moment of reflection and orientation toward
   Jerusalem-based time.

## Development Notes

JClock was independently designed and developed as part of a long personal
creative and technical process.

The app combines programming, time calculation, Hebrew and Jerusalem-centered
thought, JavaScript-based web development, and Garmin Connect IQ wearable
technology.

The project reflects a vision of time as something more than measurement: time
as orientation, meaning, and connection.

One of the main technical goals is to keep the Garmin version aligned with the
official JavaScript clock as precisely as possible, including millisecond-level
timing behavior where the device environment allows it.

Matching a wearable Garmin implementation with a JavaScript web implementation
at millisecond-level timing is a significant technical challenge, because the
two environments handle time, rendering, refresh cycles, and device limitations
differently.

A major part of the work behind JClock is making the Garmin display correspond
as closely as possible to the web-based JClock calculation, while still
respecting the technical constraints of Garmin Connect IQ devices.
